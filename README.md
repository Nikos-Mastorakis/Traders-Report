# Traders Sales Analysis

### Project Overview

 This data analysis project aims to provide insights about Net Revenue,Profit of various Suppliers of the platform

 ### Data Sources

  Traders Data: Csv files from Excel,SQL Server and Python Scripting

  ### Tools

  #### -Excel for Data Cleaning and creating measures for Gross Revenue,Net Revenue and Tax Per Product
  
   
  #### -Python for creating exchange table
  
   
  #### -SQL Server for creating dimension tables from the fact table
  
   
  #### -Power BI for Query calculations 

  Appending queries Purchases2023 and Purchases2024 into Purchases2023/24
  
   Adding Query Parameters to Sales[Order ID],to get a sample from the source lake(SQL server)
  
   Creating a function "VisualPerformance" for importng the sample above, in order to have faster performance

   
   #### -Power BI for Level Security:
   Static RLS by Country
   
   Dynamic RLS by SupplierID
   
   OLS on email column
 #### -Power BI for DAX measurements

 Yearly Profit,Profit Margin,Quarterly Profit Margin, Year-to-Date Profit and Median Sales

 Performance Analyzer for optimization of speed and efficiency
